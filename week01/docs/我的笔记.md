**maven如何使用**

 - Sync All Maven Projects ：检查 pom.xml 和 IDEA 项目结构是否一致，只做轻量同步，不会重新下载依赖。

 - Reload All Maven Projects ：完全重新解析 pom.xml，检查所有依赖，重新下载缺失或更新的 jar 包，并更新 IDEA 的依赖树。
